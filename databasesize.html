Project Description

At Cotiviti Technology, I recognized a critical operational challenge in managing storage allocation across our extensive database ecosystem. With over 330 production database systems totaling approximately 9 petabytes of data, we frequently encountered system failures due to unexpected storage exhaustion, resulting in service disruptions and emergency resource allocation.

To address this challenge, I designed and implemented an intelligent database size estimation system using machine learning techniques, specifically Long Short-Term Memory (LSTM) neural networks, to predict future storage requirements and enable proactive capacity planning.

Challenges

  • Unpredictable database growth patterns across heterogeneous systems
  • Frequent system outages due to unexpected storage exhaustion
  • Managing over 330 database instances across multiple platforms
  • High volume of emergency JIRA tickets for storage allocation
  • Reactive approach to capacity management causing business disruptions

Solutions

  • Centralized data collection system for monitoring all database instances
  • LSTM neural network model for time-series forecasting of storage usage
  • Automated alerting system for proactive capacity planning
  • Integrated workflow with storage provisioning systems
  • Predictive analytics dashboard for visualization and trend analysis

I designed a robust data collection framework that continuously gathered storage metrics from diverse database systems including Oracle, SQL Server, Vertica, and various data warehouses. The system tracked historical growth patterns, seasonal trends, and anomalies.

            connections[db_type] = []

                if db_type == 'oracle':
                    conn = self._connect_oracle(db_config)
                elif db_type == 'sqlserver':
                    conn = self._connect_sqlserver(db_config)
                elif db_type == 'vertica':
                    conn = self._connect_vertica(db_config)
                # Add other database types as needed
                connections[db_type].append((db_config['name'], conn))
        return connections

LSTM Model Implementation

The heart of the system was a Long Short-Term Memory (LSTM) neural network model that analyzed historical growth patterns to predict future storage requirements. I implemented the model using TensorFlow and Keras, optimizing for accuracy in time-series forecasting. 

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, LSTM, Dropout
from tensorflow.keras.callbacks import EarlyStopping
from sklearn.preprocessing import MinMaxScaler

def build_lstm_model(X_train, y_train, X_val, y_val):
    # Define the LSTM model architecture
    model = Sequential()
    model.add(LSTM(units=50, return_sequences=True, 
                input_shape=(X_train.shape[1], X_train.shape[2])))
    model.add(Dropout(0.2))
    model.add(LSTM(units=50, return_sequences=False))
    model.add(Dropout(0.2))
    model.add(Dense(units=25))
    model.add(Dense(units=1))
    
    # Compile the model
    model.compile(optimizer='adam', loss='mean_squared_error')
    
    # Train the model with early stopping
    early_stop = EarlyStopping(monitor='val_loss', patience=10, 
                              restore_best_weights=True)
    
    history = model.fit(X_train, y_train, epochs=100, batch_size=32,
                       validation_data=(X_val, y_val),
                       callbacks=[early_stop], verbose=1)
    
    return model, history

Predictive Analytics Engine

I developed a robust predictive analytics engine that processed the LSTM model outputs and generated actionable insights. The system provided forecasts at different time horizons (7-day, 30-day, 90-day) with confidence intervals, enabling tiered response planning. 

def generate_forecasts(model, database_data, scaler, forecast_days=[7, 30, 90]):
    """
    Generate storage forecasts for multiple time horizons
    """
    forecasts = {}
    
    for days in forecast_days:
        # Prepare input data (last 60 days of data)
        input_data = database_data[-60:].copy()
        
        # Generate predictions for each day in the forecast period
        forecast_values = []
        current_input = input_data.values.reshape(1, input_data.shape[0], input_data.shape[1])
        
        for _ in range(days):
            # Predict the next day
            next_day_scaled = model.predict(current_input)
            next_day = scaler.inverse_transform(next_day_scaled)[0][0]
            forecast_values.append(next_day)
            
            # Update input for next prediction
            # (Drop oldest day, add prediction as newest day)
            new_input = np.append(current_input[0, 1:, :], 
                                [[next_day_scaled[0][0]]], axis=0)
