Project Overview

This project was a comprehensive data visualization coursework where I created a dynamic dashboard for dummy coffee sales data, applying all major visualization theories including Tufte's visualization principles and Gestalt principles to make the dashboard more effective and intuitive.

The project encompassed the entire data analytics pipeline - from raw data preprocessing and cleaning to the development of interactive dashboards that provide clear, actionable insights for business decision-making. The focus was on creating visualizations that not only looked professional but also effectively communicated complex data patterns to stakeholders.

Visualization Theory Application

Applied fundamental visualization principles for effective data communication:

  • Tufte's Data-Ink Ratio: Maximized data representation while minimizing chart junk
  • Gestalt Proximity: Grouped related elements to show relationships
  • Color Theory: Used consistent color schemes for categorical data
  • Visual Hierarchy: Emphasized key metrics through size and positioning
  • Cognitive Load: Simplified complex visualizations for better understanding

Dashboard Architecture

Designed comprehensive dashboard with multiple visualization types:

  • Executive summary with key performance indicators (KPIs)
  • Time-series analysis for sales trends and seasonality
  • Geographic distribution maps for regional performance
  • Product category analysis with market share breakdown
  • Customer segmentation and behavior analysis
  • Interactive filters for dynamic data exploration

Business Insights & Recommendations

Generated actionable insights to improve sales and marketing strategies:

  • Identified peak sales periods and seasonal trends for inventory planning
  • Discovered high-performing product categories and underperforming segments
  • Analyzed customer purchasing patterns for targeted marketing campaigns
  • Recommended pricing strategies based on elasticity analysis
  • Provided geographic expansion opportunities based on market analysis
